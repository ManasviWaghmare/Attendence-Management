# QR-Based Student Attendance Management System 🎓

A dynamic, secure, and automated attendance system where students scan revolving QR codes on a teacher's screen.

## 🌟 Key Features
- **Dynamic QR Generation**: QR codes rotate every 10-30 seconds to prevent unauthorized sharing or photo-based spoofing.
- **AES Encryption**: QR tokens are signed and encrypted on the backend, ensuring they cannot be manually generated by students.
- **Real-time Analytics**: Teachers see live attendance counts and charts on their dashboard.
- **Automation Layer**: Background schedulers handle QR expiry and session management automatically.
- **Premium UI**: Modern dark-mode dashboard with glassmorphism and smooth transitions.

## 🏗️ Technical Architecture
- **Backend**: Spring Boot 3.2, Spring Security, Spring Data JPA, PostgreSQL.
- **Frontend**: Vanilla JS (ES6+), CSS3 (Glassmorphism), Chart.js, FontAwesome.
- **QR Engine**: Google ZXing (Java) and HTML5-QRCode (Javascript).

## 🚀 Getting Started

### 1. Prerequisites
- **Java 17+**: Ensure Java is installed and `JAVA_HOME` is set.
- **Maven 3.8+**: Required to build and run the backend.
- **PostgreSQL**: A running instance with a database named `qr_attendance`.

### 2. Backend Setup
1. Configure database credentials in `backend/src/main/resources/application.properties`.
2. Build and run the backend:
   ```bash
   cd backend
   mvn clean install
   mvn spring-boot:run
   ```
   *The server will start on `http://localhost:8080` and seed the database automatically.*

### 3. Frontend / Dashboard Setup
The frontend is a static web application that communicates with the backend API.
1. Open `frontend/pages/dashboard.html` in a modern web browser.
2. Ensure the backend is running for real-time data flow.
3. Access the dashboard to manage courses, start sessions, and view live attendance.

### 4. Student Scanner Setup
1. Open `mobile-scanner/scan.html`.
2. This mimics a mobile device interface. Enter a valid student ID (e.g., `S001`).
3. Scan the revolving QR code from the teacher's dashboard to mark attendance.
